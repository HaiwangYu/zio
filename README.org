#+title: ZIO - a high level Python and C++ interface to ZeroMQ, Zyre and others.

ZIO provides a high-level interface to ZeroMQ, Zyre and some patterns
described in the [[http://zguide.zeromq.org/][zguide]].  It also provides a /ported graph/ model for
constructing an application over a network topology.

Documentation is at https://brettviren.github.io/zio.
